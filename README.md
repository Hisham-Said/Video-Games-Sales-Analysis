Video Games Sales Analysis:
A comprehensive data analysis project that explores video game sales trends, platform performance, and industry insights using Python.
Overview:
This project analyzes video game sales data to uncover patterns and trends in the gaming industry. It generates multiple visualizations to help understand market dynamics, popular platforms, successful genres, and publisher performance.
Features:
The analysis includes five key visualizations:
Top 10 Best-Selling Video Games - Identifies the highest-grossing games globally
Platform Sales Performance - Compares total sales across different gaming platforms
Genre Average Sales Analysis - Reveals which genres perform best on average
Games Released Per Year - Tracks the volume of game releases over time
Publisher Market Share - Shows which publishers have released the most games
Dataset:
The analysis expects a CSV file with the following columns:
Name - Game title
Platform - Gaming platform (PS4, Xbox, PC, etc.)
Year - Release year
Genre - Game genre
Publisher - Publishing company
Global_Sales - Total global sales
Output:
The script produces five bar chart visualizations:
Color-coded charts for easy interpretation
Rotated labels for readability
Grid lines for precise value reading
Properly sized figures for detailed viewing
